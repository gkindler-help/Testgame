# The Home Battle: Buyer Beware

A browser-based arcade run-and-gun MVP inspired by classic side-scrollers.

## Included
- Parallax city background
- Run, jump, and shoot gameplay
- Multiple enemy types
- Required George power-up
- Final boss that is effectively unwinnable without the power-up
- HUD with budget, stress, lives, and boss armor

## Files
- index.html
- style.css
- game.js

## Run
Open `index.html` locally in a browser, or upload the folder to GitHub Pages.
